# GE - Game Engine

- A 2D C++ Game Engine with Asteroids game.

# Usage
- Open the GE.sln file in VS or Rider
- Build on x86 Configuration

# Known Issues
- Manually copy the freeglut.dll to the Debug/Release build folders
- Running the Debug/Release build independent of the IDE does not load in the audio, manual copy of the Assets folder is needed.
